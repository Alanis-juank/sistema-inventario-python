# Sistema de Gestión de Inventarios en Python

## Descripción
Aplicación desarrollada en Python utilizando Programación Orientada a Objetos para gestionar el inventario de una tienda.

## Funcionalidades
- Añadir productos
- Eliminar productos por ID
- Actualizar cantidad y precio
- Buscar productos por nombre
- Mostrar inventario completo

## Tecnologías utilizadas
- Python
- GitHub
- Programación Orientada a Objetos

## Estructura del proyecto
- producto.py → Clase Producto
- inventario.py → Gestión del inventario
- main.py → Interfaz de usuario
- README.md → Documentación
